Plate Angle & Valley Angle (Industrial Design)

Formulas

Plate angle (long side):

θL = tan⁻¹ [ H / ( (Lt − Lb) / 2 ) ]

Plate angle (width side):

θW = tan⁻¹ [ H / ( (Wt − Wb) / 2 ) ]

Valley angle:

θV = tan⁻¹ [ H / √( (ΔL/2)² + (ΔW/2)² ) ]
