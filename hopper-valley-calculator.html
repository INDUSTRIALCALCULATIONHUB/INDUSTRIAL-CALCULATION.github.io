Plate Angle & Valley Angle (Industrial Design)

Engineering Formulas

Plate angle (longitudinal sides):

θL = tan⁻¹ ( H / ΔL )
where ΔL = (Lt − Lb) / 2

Plate angle (transverse sides):

θW = tan⁻¹ ( H / ΔW )
where ΔW = (Wt − Wb) / 2

Valley angle (diagonal):

θV = tan⁻¹ ( H / √(ΔL² + ΔW²) )
